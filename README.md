## Project Description

This project analyzes an open-source software system and demonstrates Linux shell scripting skills. The software chosen for this project is VLC Media Player, an open-source multimedia player capable of playing multiple audio and video formats.

## Scripts Included

### Script 1 – System Identity Report

Displays system information such as kernel version, user name, uptime, and distribution.

### Script 2 – FOSS Package Inspector

Checks whether VLC is installed and displays information about the package.

### Script 3 – Disk and Permission Auditor

Checks important directories and reports their permissions and disk usage.

### Script 4 – Log File Analyzer

Reads a log file and counts the occurrences of a keyword such as "error".

### Script 5 – Open Source Manifesto Generator

Prompts the user with questions and generates a personalized open-source manifesto saved to a text file.

## How to Run the Scripts

Make scripts executable:

chmod +x script1.sh script2.sh script3.sh script4.sh script5.sh

Run scripts:

./script1.sh
./script2.sh
./script3.sh
./script4.sh /var/log/syslog error
./script5.sh

## Requirements

Linux system or online Linux terminal with Bash.
